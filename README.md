#  Airline Ticket Booking and Management System

##  Project Overview
This project is a web-based **Airline Ticket Booking and Management System** developed as part of the Web Programming Lab (WPL).  
It simulates a basic airline booking workflow where users can enter details, view records, and access conditional navigation features.

---

##  Features

-  **Ticket Booking Form**
  - Enter passenger name, product ID, and price  

-  **Display Records**
  - View all stored booking details  

-  **Conditional Navigation**


-  **Session Handling**
  - Data is passed between pages using sessions  

---


---

##  Technologies Used

- HTML  
- CSS  
- Python (Django)  
- SQLite  
- JavaScript (optional)

---

## How It Works

1. User enters ticket details in the form  
2. Data is stored in the database  
3. All records are displayed on the second page  
4. If ticket price > 200:
   - User is redirected to detail page  
5. Else:
   - Access is denied and user remains on the same page
